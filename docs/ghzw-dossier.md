## 二、方法论蒸馏（来自 70+ 条长文发言）

### 复盘三问（核心决策框架 · 2024-10 #19418 楼）
> "复盘就像高考的阅读理解……**涨停的股为啥涨停？它在板块的地位如何？在大盘的地位如何？都要思考的**"

### 复盘流程（2022-06 #3820 楼 · 完整自述）
1. 看湖南人复盘 → 连板股标记到通达信
2. 分析**总龙 / 高度板 / 跟风板 / 竞争板**结构
3. 预判明日走势（符合预期 / 超预期）→ **"弱转强快速板才是超预期"**
4. 多看正反两面观点 + 研报（"我是做短线的 但我花很多时间看研报"）
5. 竞争板"六分之一概率 错了太影响心态" → **不满仓搞**，盘面确认强度再上

### 选股口径
- **主线至上**："八月底以来一直在化工线里搞来搞去"（2021-09）—— 只做当期最强主线
- **逻辑硬的低位票爆发力足**：清水源（联创榜样的补涨）、冀中能源（屡屡超预期一字 + 研报基本面 + 周期主线 + 大资金喜欢大票 + 强盘口）
- **格局票 = 时代的情绪载体**："我做大众交通 是把它当**无人驾驶时代的情绪载体** 这种时代性的东西 涨三五倍很正常""**当你的认知能判断这个票能走出三五倍的时候为什么恐高 为什么不格局**"

### 风控与心性
- "每波行情赚个十五个点左右就该休息"（2018 熊市自省）
- 自认最大弱项："每次大的系统性风险都没有很好的回避"
- "交易终究拼的是心境"；"炒股快乐个嘚 全是折磨"
- 偶像：龙飞虎（"一路看着他从八十万到八百万 我现在也做到了"）· "外面的大神千千万 **我只认巴菲特**"

### 操守（2026-01 #62917 楼）
